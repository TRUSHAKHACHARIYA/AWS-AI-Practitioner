# AWS Certified AI Practitioner (AIF-C01) — Study & Practice Notes

This repository is a **well-structured documentation hub** for learning and practicing the **AWS Certified AI Practitioner (AIF-C01)** course content.

## What you’ll find here

- **Concept notes**: Clear explanations in simple language.
- **AWS service summaries**: What each service does + when to use it.
- **Exam-focused highlights**: Keywords, common pitfalls, and quick comparisons.
- **Practice section**: Mini quizzes, flashcards-style notes, and scenario questions.

## How to use this repo (recommended workflow)

- Read the topic notes in order.
- After each topic, do the related practice questions.
- Keep a running list of “confusing points” and review weekly.
