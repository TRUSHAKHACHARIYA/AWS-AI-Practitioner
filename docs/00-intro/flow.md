## Learning Flow Overview

This project follows a **step-by-step learning journey** for the AWS AI Practitioner path.

### High-level flow

1. **Dashboard**
2. **Exam Guidelines → Syllabus + Graph**
3. **Modules (1–5)**
4. **Module Content**
5. **Flashcards → Quiz (per module)**
6. **Next Module**
7. **Final Test**
8. **Results + Analytics**
9. **References + Revision (Flashcards / Notes)**

### Visual sequence

- Dashboard  
  ↓  
- Exam Guidelines → **Syllabus + Graph**  
  ↓  
- **Modules (1–5)**  
  ↓  
- Module Content  
  ↓  
- Flashcards → Quiz (for that module)  
  ↓  
- Next Module  
  ↓  
- Final Test  
  ↓  
- Results + Analytics  
  ↓  
- References + Revision (Flashcards / Notes)
